Features

  • On-demand scan with progress bar
  • Quarantine manager (delete/restore)
  • Light & Dark themes
  • Resizable, DPI-aware UI
  • Optional login screen
  • Custom status & warning labels

Installation

  1. Clone the repo: git clone https://github.com/your-username/VoidShield.git
  2. Open VoidShield.sln in Visual Studio
  3. Place VoidShield.ico in %USERPROFILE%\Documents (optional)
  4. Build & Run (F5 or Ctrl+F5)

Usage

Launch the app, log in (admin/password123), then:

  • Click Start Scan
  • Manage threats under View Quarantine
  • Tweak preferences in Settings
